UX audit · macOS + Android

A calm, legible
clipboard.

A full pass over the interface with one goal: make every screen obvious at a glance. No visual noise, no redundant decoration, no language mixing — one consistent system, sized to real macOS conventions. Below: what changed, the rules behind it, and a clickable demo of every screen and state. Toggle theme and accent in the top bar — the whole demo re-themes live.

Findings & fixes

01

Redundant type encoding

Colored tile + colored chip on every row = noise
Type now lives in one place: the colored icon tile. The kind word moves into the quiet meta line — calmer scanning, same information.
02

Mixed languages

Ukrainian labels over English data
English-only across the entire UI. One voice, predictable vocabulary, easier to localize properly later.
03

Arbitrary proportions

Guessed widths/heights, cramped rows
Locked to macOS scale: window 1040×640, sidebar 212, row 40px, popup 460. 4px spacing grid throughout.
04

Flat feed

One endless list — the "Facebook" feel
Date grouping (Pinned · Today · Yesterday · Earlier) with sticky headers. The list reads as structure, not a stream.
05

Hidden affordances

Actions only on hover, easy to miss
Clear selected state, persistent keyboard hints in footers, visible focus rings, generous hit targets (≥28px).
06

Missing states

Only the happy path was drawn
Empty, no-results, loading, multi-select, sensitive, too-large, pairing, onboarding, and four system banners — all specified.

Operating principles

◔

Content first

The clip dominates each row. Metadata recedes to 11px faint. Chrome never competes with content.

◇

Color is meaning

Color is reserved for two jobs only: the accent, and content-type coding. Nothing is colored for decoration.

≣

One of everything

One icon family, one radius scale, one type scale, one row anatomy shared by History and Popup.

⤓

Keyboard-native

Every action has a shortcut, shown in context. The mouse is optional, never required.

◐

Accessible by default

WCAG AA contrast, type never coded by color alone (icon + word), focus-visible, reduced-motion respected.

○

Quiet surfaces

Subtle borders over heavy fills, restrained shadows, lots of negative space. Calm beats busy.

Information architecture

Where blocks live — and when they appear

The main window is a fixed three-zone frame. Persistent chrome on the left and bottom; everything else swaps inside the content zone. Transient blocks (banners, bulk bar, modals) appear only when their condition is true.

Onboarding — three steps, then out of the way

The one hard requirement on macOS is the Accessibility permission needed to paste into the focused app. Explain why before asking; confirm the grant live.

STEP 1 / 3

Welcome to CopyPaste

Everything you copy is saved here, searchable and ready to paste. Press ⌥⌘V any time to open the popup.

Grant Accessibility access

CopyPaste needs this one permission to paste into the app you're using. It never reads anything else.

You're all set

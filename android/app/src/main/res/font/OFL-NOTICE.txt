Bundled fonts — SIL Open Font License 1.1
==========================================

This directory bundles the following font binaries, used by the Compose UI
per DESIGN-SYSTEM-v2 §1/§10 for pixel-consistent typography across platforms:

  inter_regular.ttf      Inter, weight 400
  inter_medium.ttf       Inter, weight 500
  inter_semibold.ttf     Inter, weight 600
  jetbrains_mono_regular.ttf   JetBrains Mono, weight 400
  jetbrains_mono_medium.ttf    JetBrains Mono, weight 500
